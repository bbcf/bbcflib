Function Details [hide private]

save_motif_profile(ex, motifs, background, genrep, chromosomes, data_path, keep_max_only=False, description='', threshold=0, via='lsf')

 

Scan a set of motifs on a set of regions and saves the results as an sql file. The 'motifs' argument is a dictionary with keys motif names and values PWM with 'n' rows like: "1 p(A) p(C) p(G) p(T)" where the sum of the 'p's is 1 and the first column allows to skip a position with a '0'.
